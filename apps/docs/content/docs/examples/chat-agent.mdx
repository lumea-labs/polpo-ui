# Chat Agent

A complete example of a chat page connected to a Polpo agent.

```tsx
import { Chat } from "@polpo-ai/chat";
import { PolpoProvider } from "@polpo-ai/react";

export default function ChatPage() {
  return (
    <PolpoProvider baseUrl="https://api.polpo.sh" apiKey="sk_live_...">
      <div className="h-screen">
        <Chat agent="coder" sessionId="session_abc" />
      </div>
    </PolpoProvider>
  );
}
```

## What you get

- Streaming responses with live markdown rendering
- Tool call chips (bash, read, write, search, email, http)
- Session persistence across page reloads
- Auto-scroll with scroll-to-bottom button
- Keyboard shortcuts (Enter to send, Shift+Enter for newline)
- File drag-and-drop detection
- Loading skeletons while connecting

## Customization

For full control over the layout, use the composable API:

```tsx
import {
  ChatProvider,
  ChatMessages,
  ChatSkeleton,
} from "@polpo-ai/chat";
import { useSubmitHandler } from "@polpo-ai/chat/hooks";

function CustomChat() {
  const { inputRef, handleSubmit, handleKeyDown } = useSubmitHandler({
    onSubmit: (text) => {/* send message */},
  });

  return (
    <ChatProvider agent="coder" sessionId="session_abc">
      <ChatMessages className="flex-1" />
      <form onSubmit={handleSubmit}>
        <input
          ref={inputRef}
          onKeyDown={handleKeyDown}
          placeholder="Message coder..."
        />
      </form>
    </ChatProvider>
  );
}
```
