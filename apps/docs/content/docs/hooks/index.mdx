# Hooks

## useSubmitHandler

Handles message submission with keyboard shortcuts and form state.

```tsx
import { useSubmitHandler } from "@polpo-ai/chat/hooks";

const { inputRef, handleSubmit, handleKeyDown } = useSubmitHandler({
  onSubmit: (text) => sendMessage(text),
});
```

## useDocumentDrag

Detects document-level drag events for file upload UX.

```tsx
import { useDocumentDrag } from "@polpo-ai/chat/hooks";

const isDragging = useDocumentDrag();

return (
  <div className={isDragging ? "border-dashed border-primary" : ""}>
    Drop files here
  </div>
);
```

## useChatContext

Access the chat provider context from any child component.

```tsx
import { useChatContext } from "@polpo-ai/chat";

const { agent, messages, isStreaming, sessionId } = useChatContext();
```
