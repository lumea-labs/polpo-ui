# ChatProvider

Provides chat context to all child components. Use this when you want to compose your own chat layout.

## Import

```tsx
import { ChatProvider, useChatContext } from "@polpo-ai/chat";
```

## Usage

```tsx
<ChatProvider agent="coder" sessionId="session_abc">
  <div className="flex flex-col h-[600px]">
    <ChatMessages className="flex-1" />
    <ChatInput />
  </div>
</ChatProvider>
```

## Accessing context

```tsx
function CustomHeader() {
  const { agent, sessionId, messages, isStreaming } = useChatContext();

  return (
    <div>
      Chatting with {agent} — {messages.length} messages
      {isStreaming && " (streaming...)"}
    </div>
  );
}
```
