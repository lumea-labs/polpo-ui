# Chat

Composable chat components with three levels of control.

| Component | Description |
|-----------|-------------|
| [`Chat`](/docs/components/chat/chat) | Full chat interface — zero config |
| [`ChatProvider`](/docs/components/chat/chat-provider) | Context provider for chat state |
| [`ChatMessages`](/docs/components/chat/chat-messages) | Virtualized message list |
| [`ChatMessage`](/docs/components/chat/chat-message) | Single message renderer |
| [`ChatSkeleton`](/docs/components/chat/chat-skeleton) | Loading skeleton |
