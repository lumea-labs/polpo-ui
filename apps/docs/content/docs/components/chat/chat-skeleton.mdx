# ChatSkeleton

Skeleton loading states that match the chat layout.

## Import

```tsx
import { ChatSkeleton, MessageSkeleton, UserMessageSkeleton } from "@polpo-ai/chat";
```

## Usage

```tsx
{isLoading ? <ChatSkeleton /> : <ChatMessages />}
```

## Variants

- `ChatSkeleton` ‚Äî Full chat skeleton (multiple messages)
- `MessageSkeleton` ‚Äî Single assistant message skeleton
- `UserMessageSkeleton` ‚Äî Single user message skeleton
