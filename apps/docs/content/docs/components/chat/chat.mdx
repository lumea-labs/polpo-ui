# Chat

The `Chat` component provides a complete conversational interface with streaming, tool call rendering, and session persistence.

## Import

```tsx
import { Chat } from "@polpo-ai/chat";
```

## Usage

### Basic

```tsx
<Chat agent="coder" sessionId="session_abc" />
```

### With custom styling

```tsx
<Chat
  agent="coder"
  sessionId="session_abc"
  className="h-[600px] border rounded-lg"
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `agent` | `string` | — | Agent name to chat with (required) |
| `sessionId` | `string` | — | Session ID for conversation continuity |
| `className` | `string` | — | Custom CSS class |
| `placeholder` | `string` | `"Message..."` | Input placeholder text |
| `onMessage` | `(msg) => void` | — | Callback when a message is sent |

## Composition

`Chat` is a convenience wrapper around:

```tsx
<ChatProvider agent={agent} sessionId={sessionId}>
  <ChatMessages />
  <ChatInput />
</ChatProvider>
```

For full control, use these components individually. See [ChatProvider](/docs/components/chat-provider).
