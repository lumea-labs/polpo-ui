# Getting Started

## Installation

### Via npm

```bash
npm install @polpo-ai/chat @polpo-ai/sdk @polpo-ai/react
```

Peer dependencies:

```bash
npm install react react-virtuoso lucide-react streamdown
```

### Via shadcn/ui registry

```bash
npx shadcn@latest add "https://ui.polpo.sh/registry/polpo-chat"
```

This copies the components into your project. You own the code.

## Setup

Wrap your app with the Polpo provider:

```tsx
import { PolpoProvider } from "@polpo-ai/react";

function App({ children }) {
  return (
    <PolpoProvider baseUrl="https://api.polpo.sh" apiKey="sk_live_...">
      {children}
    </PolpoProvider>
  );
}
```

## First Chat

```tsx
import { Chat } from "@polpo-ai/chat";

function ChatPage() {
  return <Chat agent="coder" sessionId="session_abc" />;
}
```

That's it. You have a full conversational interface with streaming, tool calls, and session persistence.
