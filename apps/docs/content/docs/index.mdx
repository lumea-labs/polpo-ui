# @polpo-ai/ui

UI components for building AI agent interfaces with [Polpo](https://polpo.sh). Built on React + Tailwind CSS + shadcn/ui.

## What is this?

A component library that gives you production-ready chat interfaces, tool renderers, and agent views — wired to the Polpo SDK out of the box.

Three levels of composition:

- **Zero config** — Drop in `<Chat agent="coder" />` and get a full conversational UI
- **Customizable** — Use `ChatProvider` + individual components to control layout
- **Headless** — Use hooks only (`useSubmitHandler`, `useDocumentDrag`) and build your own UI

## Packages

| Package | Description |
|---------|-------------|
| `@polpo-ai/chat` | Chat components, tool renderers, hooks |
| `@polpo-ai/ui` | CLI for installing components (coming soon) |

## Quick Start

```bash
npm install @polpo-ai/chat @polpo-ai/sdk @polpo-ai/react
```

```tsx
import { Chat } from "@polpo-ai/chat";

function App() {
  return <Chat agent="coder" sessionId="session_abc" />;
}
```

Or install via shadcn registry:

```bash
npx shadcn@latest add "https://ui.polpo.sh/registry/polpo-chat"
```
